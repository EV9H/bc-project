# bc-project

## front-end
- Framework: React 
- To start server: 
    npm start 
- *UI framework
    antd (npm install antd) 


    
## back-end 
- Framework: django
- Database: Postgresql 
- Instructions: 
    - create virtual env and activate
    - install requirements: 
        pip install -r requirements.txt
    - to run server, under the same directory of manage.py: 
        python manage.py runserver
